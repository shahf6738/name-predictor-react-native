# Name Predictor - React Native

A React Native version of my Name Prediction Game originally built with HTML, CSS and JavaScript.

## Features

- Multi-screen game flow
- React Navigation
- React Context API
- TypeScript
- Progress tracking
- Undo selections
- Prediction reveal screen

## Tech Stack

- React Native
- Expo
- TypeScript
- React Navigation

## Screens

- Welcome
- Name Length
- Column Selection
- Row Selection
- Prediction Result

## Getting Started

```bash
npm install
npm start
```
